# mySkills

个人 AI skill 仓库。

这个仓库用于存放可复用的 AI skill，包括 skill 说明、代理入口配置、配套脚本、参考资料和示例文件。

## 仓库结构

- `chrome-cdp/`：本地 Chrome 调试与交互 skill
- `cn-commit/`：中文规范提交 skill
- `code-reviewer/`：代码审查 skill
- `context-compression/`：上下文压缩 skill
- `full-auto-executor/`：高自治执行 skill
- `gh-guide/`：GitHub CLI 使用规范 skill
- `safe-auto-executor/`：安全边界内自动执行 skill
- `docs/`：设计与计划文档
